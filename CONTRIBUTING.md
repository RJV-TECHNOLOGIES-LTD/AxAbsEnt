# Contributing to Super_SQL

We welcome contributions to expand the unified model and its recursive substrate.

## Contribution Guidelines
- All child dimensions must be validated via the AI system or notebooks.
- Use the provided fusion engine to combine parent domains into new models.
- Ensure paradox handling and recursion flags are respected.
- All `.tex` files must compile without LaTeX errors.

## Pull Requests
- Submit from a feature branch.
- Reference affected dimensions or domain IDs.
- Include test examples and `.ipynb` runs where relevant.

Thank you for expanding the recursive system.
