# 📜 Code of Conduct

## 👁️‍🗨️ Overview

This project — **AxAbsEnt: Enhanced Mathematical Ontology of Absolute Nothingness** — is a frontier of scientific truth. It welcomes contributors who respect rigorous logic, ontological coherence, and absolute integrity in discourse.

All participants in the repository — including contributors, reviewers, maintainers, and external collaborators — are expected to uphold the principles of **intellectual honesty**, **scientific clarity**, and **constructive collaboration**.

We do not tolerate **toxicity**, **academic arrogance**, **factually false corrections**, or any behavior that undermines the **precision and reproducibility** of the unified framework.

---

## 🧠 Core Principles

1. **Respect the Ontology**  
   All discussions and contributions must respect the foundational constructs of the AxAbsEnt model. Attempts to rewrite core logic without formal disproof will be rejected.

2. **Be Precise and Falsifiable**  
   Claims must be supported by mathematical evidence, empirical reproducibility, or logical validity.

3. **Zero Intellectual Abuse**  
   There is no room for gaslighting, elitism, dismissive commentary, or malicious PRs designed to discredit the model or its author.

4. **No Ideological Contamination**  
   This is not a forum for philosophical, political, or social agendas. Scientific rigor only.

5. **Communicate with Clarity**  
   Use precise, structured language. Avoid vague terms like “I think” or “maybe.”

---

## 🚫 Unacceptable Behavior

- Personal attacks, name-calling, or intimidation
- False flagging of valid scientific work as “pseudoscience”
- Submitting pull requests that introduce inaccuracies, regression, or covert sabotage
- Using the issue tracker to vent, spam, or derail scientific progress
- Misrepresenting the theory in public-facing discourse without evidence

---

## ✅ Acceptable Behavior

- Respectfully submitting issues with reproducible cases
- Opening discussions that follow scientific method
- Proposing theory extensions grounded in mathematical logic
- Helping others understand AxAbsEnt concepts clearly and respectfully
- Upholding the integrity of theoretical evolution
